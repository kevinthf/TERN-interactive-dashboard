## Data Source
TERN is Australia's national ecosystem observatory, collecting long-term measurements of terrestrial ecosystems across Australia and storing these observations in its databases.
By leveraging this rich dataset, we can explore the data and uncover interesting patterns, trends, and insights from the raw datasets.

The datasets were acquired from the [TERN EcoPlots](https://ecoplots.tern.org.au/) platform, which hosts a wide range of environmental and ecological data. 

The detailed procedure for downloading raw data from TERN EcoPlots can be found [here](https://ternaus.atlassian.net/wiki/spaces/TERNSup/pages/2330394627/TERN+EcoPlots+User+Guide).



## Data Cleaning
The Data_preprocessing R script mainly conducting data cleaning and summarizing to produce output csv file for dashboard presentation.

Here we use three datasets including "plant occurrence", "weather" and "soil profile" to perform the following processing and visualisations:

1. plant occurrence- Dataset containing observations of plant species (identified by their scientific names) recorded within surveyed sites. <br/> We aggregate these observations by survey site and survey period to generate summaries of
  - species richness
  - taxonomic abundance (family, genus, species) 
2. weather- Dataset containing weather attributes of surveyed sites
  - precipitation
  - solar radiation
  - temperature (max, min, mean)
3. soil profile- Dataset containing soil attributes collected at surveyed sites
  - soil pH
  - soil electrical conductivity
  - soil texture

The three datasets are then joined to investigate the relationships and correlations between the variables.

If you are interested, you can also download other datasets from the [TERN EcoPlots](https://ecoplots.tern.org.au/) and modify the code to generate other datasets.<br/> Make sure they are saved in the rawdata folder to run the preprocessing script.

🔴**This step is not required if you feel too overwhelmed.** <br/>
